# ClearViewableVariables

## Description
Clears all loaded viewable variables.

## Parameters
None.

## Return Value
None.

## Syntax (C)
```RSDK.ClearViewableVariables()```

## Syntax (C++)
```Dev::ClearViewableVariables()```

## Examples (C)
- ```RSDK.ClearViewableVariables();```

## Examples (C++)
- ```Dev::ClearViewableVariables();```
