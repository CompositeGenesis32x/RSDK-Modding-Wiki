# ProcessAnimation

## Description
Processes the animation applied to an animator.

## Parameters

- `animator`
The animator used to handle animation of the frames.

## Return Value
None

## Syntax (C)
```RSDK.ProcessAnimation(Animator *animator)```

## Syntax (C++)
```animator.Process()```

## Examples (C)
- ```RSDK.ProcessAnimation(&animator);```

## Examples (C++)
- ```animator.Process();```
