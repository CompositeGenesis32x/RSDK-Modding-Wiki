# LoadOnlineMenu

## Description
Sets the entry of the leaderboard with index of `ID` to `Result`.

## Parameters
- `ID`
The ID of the leaderboard to edit.
- `Result`
The new leaderboard record.

## Return Value
None.

## Syntax
```
SetLeaderBoard(int ID, int Result)
```

## Examples
```
SetLeaderBoard(TempValue0, TempValue1)
```
