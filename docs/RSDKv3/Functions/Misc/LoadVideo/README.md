# LoadVideo

## Description
Loads a video with the name of `VideoName`.

## Parameters
- `VideoName`
The name of the video, from path `Videos/`

## Return Value
None.

## Syntax
```
LoadVideo("string VideoName")
```

## Examples
```
LoadVideo("Opening")
```

## Notes
The supported video formats are .rsv and .ogv, the original mobile version uses .mp4
