# Sin

## Description
Gets the value from lookup table `Sin512` based on `Angle` and sets it in `Store`.

## Parameters
- `Store`
The variable to store the result.
- `Angle`
The value of angle.

## Return Value
None.

## Syntax
```
Sin(var Store, int Angle)
```

## Examples
```
Sin(Object.Value3, TempValue0)
```
