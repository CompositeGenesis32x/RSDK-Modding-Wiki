# Cos256

## Description
Gets the value from lookup table `cos256` based on `angle` and sets it in `store`.

## Parameters
- `store`
The variable to store the result.
- `angle`
The value of angle.

## Return Value
None.

## Syntax
```
Cos256(var store, int angle)
```

## Examples
```
Cos256(temp0, object.value2)
```
