# Rand

## Description
Gets a random value from 0 to `max`(not included) and stores it in `store`.

## Parameters
- `store`
Variable from where to store the result.
- `max`
Exclusive maximum for the random value to get.

## Return Value
None.

## Syntax
```
Rand(var store, int max)
```

## Examples
```
Rand(object.value3, 8)
```
