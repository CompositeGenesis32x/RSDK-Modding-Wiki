# SetTableValue

## Description
Sets value on `table` at `index` to `value`

## Parameters
- `value`
`index` new value
- `index`
Tables's index to set
- `table`
Name of the table from where to set the value

## Return Value
None.

## Syntax
```
SetTableValue(int value, int index, table)
```

## Examples
```
SetTableValue(55, 3, TestObject_Table)
```
