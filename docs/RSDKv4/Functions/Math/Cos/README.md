# Cos

## Description
Gets the value from lookup table `cos512` based on `angle` and sets it in `store`

## Parameters
- `store`
The variable to store the result
- `sngle`
The value of angle

## Return Value
None.

## Syntax
```
Cos(var store, int angle)
```

## Examples
```
Cos(temp0, object.value2)
```
