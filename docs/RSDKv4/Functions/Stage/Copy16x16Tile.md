# Copy16x16Tile

## Description
Copies the tileset image data of `scrTile` to `destTile`.

## Parameters
- `destTile`
The tile that will be overwritten.
- `scrTile`
The tile that will be copied.

## Return Value
None.

## Syntax
```
Copy16x16Tile(int destTile, int scrTile)
```

## Example
```
Copy16x16Tile(174, 173)
```

## Notes
Used for animated tiles.
