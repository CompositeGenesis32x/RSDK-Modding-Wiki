# NOTIFY_ADD_COIN

## Description
Awards the player the given amount of coins.

## Parameters
- `count`
The number of coins to give the player.

## Return Value
None.

## Syntax
```
CallNativeFunction2(NotifyCallback, NOTIFY_ADD_COIN, int count)
```

## Examples
```
CallNativeFunction2(NotifyCallback, NOTIFY_ADD_COIN, 1)
```
