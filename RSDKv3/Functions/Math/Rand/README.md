# Rand

## Description
Gets a random value from 0 to `Max`(not included) and stores it in `Store`

## Parameters
- `Store`
The variable to store the result
- `Max`
The exclusive maximum for the random value to get

## Return Value
None.

## Syntax
```Rand(var Store, int Max)```

## Examples
- ```Rand(Object.Value3, 8)```
