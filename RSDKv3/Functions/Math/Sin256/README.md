# Sin256

## Description
Gets the value from lookup table `sin256` based on `Angle` and sets it in `Store`

## Parameters
- `Store`
The variable to store the result
- `Angle`
The value of angle

## Return Value
None.

## Syntax
```Sin256(var store, int angle)```

## Examples
- ```Sin256(Object.Value3, TempValue0)```
