# CopyPalette

## Description
Copies `SrcPal` into `DestPal`.

## Parameters
- `SrcPal`
The palette bank to copy, indices 0-7 are valid
- `DestPal`
The palette bank to paste, indices 0-7 are valid

## Return Value
None.

## Syntax
```CopyPalette(int ScrPal, int DestPal)```

## Examples
- ```CopyPalette(0, 7)```
